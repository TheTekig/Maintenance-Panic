<h2>🧠 About the Game</h2>

<p>
<b>Maintenance Panic</b> is a chaotic top-down action/strategy game where you play as a modular maintenance robot sent to stabilize a collapsing industrial factory.
</p>

<p>
The factory is failing. Machines are breaking. Systems are shutting down. Production must not stop.
</p>

<p><b>You are the last active unit.</b></p>

<hr/>

<h2>🎯 Core Concept</h2>

<ul>
  <li>⚙️ Maintain production under pressure</li>
  <li>🔧 Equip limited tools to solve problems</li>
  <li>🪝 Use a grappling system for movement</li>
  <li>📦 Maximize box production efficiency</li>
  <li>🚨 Survive increasing system chaos</li>
</ul>

<hr/>

<h2>🔁 Gameplay Loop</h2>

<ol>
  <li>Machines fail 🚨</li>
  <li>Move through the factory 🏃</li>
  <li>Switch tools 🔄</li>
  <li>Fix systems 🔧</li>
  <li>Maintain production 📦</li>
  <li>Chaos escalates 💥</li>
</ol>

<hr/>

<h2>🎮 Mechanics</h2>

<h3>Movement</h3>
<ul>
  <li>Top-down WASD movement</li>
  <li>Grapple-based mobility system</li>
</ul>

<h3>Tools System</h3>
<ul>
  <li>3 active slots only</li>
  <li>Instant switching</li>
</ul>

<table border="1" cellpadding="8">
<tr>
<th>Tool</th>
<th>Function</th>
</tr>

<tr>
<td>🔧 Repair Tool</td>
<td>Fix broken machines</td>
</tr>

<tr>
<td>⚡ Energy Tool</td>
<td>Restore power systems</td>
</tr>

<tr>
<td>🧲 Magnet</td>
<td>Pull objects and items</td>
</tr>
</table>

<hr/>

<h2>🏭 Factory System</h2>

<ul>
  <li>Machines generate boxes 📦</li>
  <li>Conveyor belts transport them</li>
  <li>Failures interrupt production</li>
  <li>Player must react quickly</li>
</ul>

<hr/>

<h2>📊 Scoring System</h2>

<ul>
  <li><b>Primary:</b> Boxes per minute (efficiency)</li>
  <li><b>Secondary:</b> Total boxes delivered</li>
</ul>

<hr/>

<h2>🤖 Enemies</h2>

<ul>
  <li>Small sabotage robots 🐀</li>
  <li>Disrupt machines and systems</li>
  <li>Increase chaos dynamically</li>
</ul>
